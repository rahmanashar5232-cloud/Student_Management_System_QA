# Student Management System - QA Assessment Project

## 📌 Project Overview
This repository contains the complete QA Assessment deliverables for validating the backend services, database integrity, and front-end flows of the Student Management System.

## 📁 Repository Structure & Deliverables Covered
- `Documentation` Contains the formal Test Plan and a professional Bug Report in tabular format.
- `API_Testing` Contains the exported Postman Collection JSON covering User Registration, Login, and CRUD validations.
- `DB_Testing` Contains SQL Validation Scripts checking constraints, joins, duplicate records, and data integrity.
- `Automation` Implements the Page Object Model (POM) framework using Python & Pytest, mapping out 10 critical test cases (Login, Registration, Search, Update, Delete, Logout).
